A dynamic web application designed to serve as a database of recipes. Users can
explore meal options and search for recipes that include specific ingredients. Built
with a focus on user-friendly navigation, the app allows users to easily find recipes
and ingredients, making meal planning more efficient. The backend is made using
Spring Boot and the frontend is made using React. To load data on the frontend, the backend URL must be accessed.
https://food-application-f4xc.onrender.com (Backend)
https://food-application-frontend.onrender.com (Frontend)
